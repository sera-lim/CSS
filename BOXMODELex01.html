Box Model Exercise 01

  • h3 태그는 대표적인 block level element 이고, a 태그는 대표적인 inline element 이다.
    두 태그의 테두리에 다음과 같이 표시하면 한 눈에 파악할 수 있다.

    • h3, a {
    border-width:5px;
    border-color:red;
    border-style:solid;
    }

    • 의미를 가지지 않으면서 단순히 영역을 구분하는 태그가 2개 있는데, div와 span다.
    div는 block level element, span은 inline element라는 점이 다르다.
